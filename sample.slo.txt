//searching multiple value
awk '/Raju|Sham/{print $0}' sample.txt

// ignore case while searching

awk 'BEGIN{IGNORECASE=1} /raju/{print $0}' sample.txt // not run  my pc because Some systems use mawk instead of gawk, and IGNORECASE does not work there.

Your system may not support IGNORECASE. Use this instead:
awk 'tolower($0) ~ /raju/' sample.txt    // ~ means pattern matching operator.
